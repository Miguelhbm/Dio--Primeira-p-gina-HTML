Estruturas Básicas do HTML

(Voltar)

HTML (Hypertext Markup Language) é uma linguagem de marcação utilizada para criar a estrutura e o conteúdo de páginas da web. Ele fornece uma maneira de estruturar informações para serem exibidas em um navegador da web, incluindo texto, imagens, links, formulários e outros elementos multimídia.

O HTML é uma linguagem de marcação, não de programação, o que significa que seu principal objetivo é descrever a estrutura e o conteúdo de uma página da web, em vez de fornecer funcionalidades interativas ou de programação. Para isso, são utilizadas outras tecnologias, como CSS (Cascading Style Sheets).

Quando um navegador da web interpreta um documento HTML, ele renderiza o conteúdo da página com base nas instruções fornecidas pelas tags HTML, exibindo o texto, imagens e outros elementos conforme especificado no código.

Você escreve o código HTML em um arquivo de texto usando um editor de texto simples, como o Bloco de Notas no Windows ou o TextEdit no Mac. Este arquivo geralmente tem uma extensão ".html" ou ".htm".

O código HTML é composto por uma série de elementos, cada um deles representando diferentes partes do conteúdo da página. Cada elemento é identificado por uma tag, que é colocada entre colchetes angulares "<>".

Tags

(Voltar)

Tags no HTML são elementos que definem a estrutura e o conteúdo de uma página da web. Elas são usadas para marcar diferentes partes do texto para indicar ao navegador como ele deve ser exibido.

Cada tag HTML possui uma função específica e pode conter atributos que fornecem informações adicionais sobre como o elemento deve ser tratado.

A maioria das tags HTML tem uma tag de abertura e uma tag de fechamento. A tag de abertura é usada para iniciar o elemento e a tag de fechamento é usada para encerrá-lo. Exemplos de Tags:

  • HTML- Abre e fecha um HTML.
  • Head- primeiras informações que o navegante verá na página.
  • Body- todas as informações que o navegante verá na página.
  • I- italico; Strong- negrito e U- sobrinhar.

As tags HTML são fundamentais para estruturar uma página da web e são a base para a criação de conteúdo na internet. Elas permitem que os desenvolvedores especifiquem diferentes tipos de conteúdo, como texto, imagens, links, listas e muito mais.

Atributos das Tags

(Voltar)

Atributos são informações adicionais que podem ser incluídas em tags HTML para fornecer mais detalhes sobre como um elemento deve ser tratado ou exibido. Eles são usados para modificar ou fornecer informações adicionais sobre um elemento HTML. Os atributos são sempre especificados dentro da tag de abertura de um elemento e são escritos como pares nome/valor, separados por um sinal de igual (=) e envolvidos por aspas duplas ("").

Muitas outras tags HTML têm atributos específicos que podem ser usados para personalizar o comportamento ou a aparência do elemento. Por exemplo, a tag (para links) possui um atributo href que especifica o URL de destino do link.

Os atributos são uma parte importante da criação de páginas da web, pois fornecem flexibilidade e controle sobre como o conteúdo é apresentado e interage com os usuários. Exemplos de atributos de tags:

  • href- Especificar o destino do Link
  • style- Permite definição de estilo para um elemento específico
  • title- Informações adicionais sobre o elemento
